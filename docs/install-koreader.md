# Installing KOReader on Kobo Clara BW

1. Download the latest KOReader release from [koreader.rocks](https://koreader.rocks)
   - Choose the **Kobo** build (`.zip`)
2. Connect Kobo via USB
3. Extract the `.zip` — copy the `.adds` folder to the root of the Kobo drive
4. Safely eject and reboot the Kobo
5. Launch KOReader from the Kobo home screen (it appears as a book called "KOReader")

**Device info:** Kobo Clara BW · codename `spaBWTPV`
